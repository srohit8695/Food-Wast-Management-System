# Food-Wast-Management-System
This is a simple food waste management system using local ROOM database, different homescreen for both food donor and acceptor and enabled view binding. Google map integration is also done for tracking the location for donor, acceptor and to route.A portion is developed for food tracking and multiple options are also given like food status, accept food and also cancel the accepted food and moves back to the common view.Separate history is also maintained for both donor and acceptor.
